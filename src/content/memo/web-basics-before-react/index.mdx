# React·Next.js 전에 정리한 웹 기초

프레임워크 문법을 외우기 전에, 최소한 이 정도는 알고 있자는 마음으로 정리한 메모다.

## 1. HTML: 구조(Structure)

- 문서의 뼈대를 만든다.
- `header`, `main`, `section`, `article`, `footer` 같은 **시멘틱 태그**를 기억해 두면 좋다.

## 2. CSS: 표현(Style)

- 레이아웃과 색, 폰트 등을 담당한다.
- 핵심 키워드 몇 개
  - `display` (`block`, `inline`, `flex`, `grid`)
  - `margin`, `padding`
  - `color`, `background`, `font-size`

## 3. JavaScript: 행동(Behavior)

- 사용자 입력, 이벤트, 비동기 통신 등을 처리한다.
- 버튼 클릭, 폼 전송, API 호출 등 **인터랙션의 중심**.

## 4. DOM(Document Object Model)

- 브라우저가 HTML을 트리 구조로 표현한 것.
- React는 결국 **DOM을 편하게 다루기 위한 도구**라는 관점이 중요하다.

이 정도를 몸에 익혀 두면,  
React와 Next.js를 배울 때 “왜 이렇게 설계되어 있는지”가 훨씬 잘 보인다.
