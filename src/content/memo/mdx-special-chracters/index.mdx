# 🛠 MDX 특수문자 모음집

MDX에서 문법 충돌이 나는 특수문자들을 위한 HTML Entity 코드 모음입니다.

## 1. 화살표 (Flow & Logic)

가장 많이 깨지는 녀석들입니다. `->` 대신 아래 코드를 쓰세요.

| 모양 | 이름 | HTML 코드 (추천) | 설명 |
| :---: | :--- | :--- | :--- |
| &rarr; | Right Arrow | `&rarr;` | 기본 화살표 (`->` 대용) |
| &larr; | Left Arrow | `&larr;` | |
| &harr; | Left-Right | `&harr;` | 양방향 |
| &rArr; | Double Right | `&rArr;` | `=>` 함수나 함의 표현 |
| &#8674; | Dotted Right | `&#8674;` | 점선 화살표 (비동기 흐름 등) |
| &#8627; | Down Right | `&#8627;` | 하위 디렉토리 구조 표현 |

## 2. MDX 4대장 (필수 이스케이프)

이 문자들은 MDX가 코드로 인식할 수 있어 **반드시** 코드로 쓰는 게 안전합니다.
*(모양 컬럼의 문자들도 안전하게 Entity로 변환했습니다)*

| 모양 | 이름 | HTML 코드 | 위험성 |
| :---: | :--- | :--- | :--- |
| &lt; | Less Than | `&lt;` | HTML 태그 열기로 오인 |
| &gt; | Greater Than | `&gt;` | HTML 태그 닫기/인용문으로 오인 |
| &#123; | Open Brace | `&#123;` | JS 변수 시작으로 오인 |
| &#125; | Close Brace | `&#125;` | JS 변수 끝으로 오인 |

| &#8984; | Command | `&#8984;` | |
| &#8997; | Option (Alt) | `&#8997;` | |
| &#8679; | Shift | `&#8679;` | |
| &#8963; | Control | `&#8963;` | |
| &#8682; | Caps Lock | `&#8682;` | |
| &#9166; | Enter/Return | `&#9166;` | |
| &#9003; | Delete | `&#9003;` | |
| &#9099; | Esc | `&#9099;` | |

## 4. 수학 & 개발 기호

| 모양 | 이름 | HTML 코드 | 비고 |
| :---: | :--- | :--- | :--- |
| &ne; | Not Equal | `&ne;` | `!=` 대신 깔끔하게 |
| &le; | Less Equal | `&le;` | `<=` 대신 |
| &ge; | Greater Equal | `&ge;` | `>=` 대신 |
| &times; | Multiply | `&times;` | `x` 대신 곱하기 기호 |
| &copy; | Copyright | `&copy;` | 푸터용 |
| &hellip; | Ellipsis | `&hellip;` | 점 3개 깔끔하게 |
