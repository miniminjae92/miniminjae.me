# Obsidian 노트 자동 커밋·동기화 설정

노트를 자주 쓰지만 커밋·푸시를 자꾸 까먹어서, 자동화 플러그인 설정을 메모해 둔다.

## 1. Auto commit interval

- 예: 10분 또는 30분
- 이 시간 동안 변경이 없으면 커밋이 일어나지 않는다.
- “타이핑 멈추고 잠깐 쉴 때” 커밋이 실행되는 느낌으로 사용.

## 2. Auto commit after latest commit

- ON이면, 수동 커밋을 해도 이후 자동 커밋이 계속 동작한다.
- “중요한 시점에 수동 커밋 + 평소엔 자동 커밋” 조합이 가능하다.

## 3. Auto pull

- 1분 간격처럼 짧게 두어도 대부분은 **체크만** 한다.
- 다른 기기에서 수정한 내용을 빠르게 가져오기 위한 옵션.

## 4. 알림

- 알림이 시끄럽다면 플러그인 옵션에서 notification을 끈다.

이 메모는 “한 번 세팅해 두고 잊어버리지 않기 위한 기억 장치”다.
